# 5minsbliss Content Ops

Content operations workspace for 5minsbliss:
- New series production (from 2026-05)
- Legacy content repurposing (IG carousel + YouTube)
- QA, publishing, and KPI tracking

## Structure
- `series/2026-05-strategic-serenity/` — new series pipeline
- `repurpose/` — IG/YT repurpose workflow
- `assets/` — image and template assets
- `ops/` — SOP, style guide, QA checklist
- `data/` — post index and metadata

2. Repurpose longform -> IG carousel + YT script
3. Weekly KPI review
