# Dialogue Demo Package (Podcast-style)

## Format
- Two-person dialogue: HOST + COACH
- Voice: Edge TTS (en-US-GuyNeural, en-US-JennyNeural)
- Visual: static themed background + burned subtitles

## Files
- `dialogue-script.txt`
- `voiceover.mp3`
- `captions.srt`
- `final-dialogue-demo.mp4`

## Subtitle strategy
- Speaker labels per line (`HOST:` / `COACH:`)
- Short lines for mobile readability
- Bottom-centered, high-contrast styling
