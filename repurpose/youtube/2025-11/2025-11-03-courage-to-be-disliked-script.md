# YouTube Script - Courage to Be Disliked at Work

Source: https://5minsbliss.com/the-courage-to-be-disliked-at-work-holding-boundaries-speaking-truth-in-professional-spaces/
Target length: 6-8 min

## Hook (0:00-0:20)
你有沒有在會議上明明有想法，卻怕被討厭所以吞回去？
今天我們聊：在職場，如何「不討好」但依然被尊重。

## Core
- 為何過度討好會讓你失去專業信用
- 3步驟：價值澄清、理性表達、承受分歧
- 一週可執行挑戰

## 7-Day Challenge
Day1 寫3個原則
Day2-4 練習句型（我觀察到…我建議…）
Day5 在低風險場合發言
Day6 檢討反應
Day7 調整策略

## CTA
如果你也在學習「溫和而堅定」的表達，留言打：我選擇尊重，不再過度討好。
