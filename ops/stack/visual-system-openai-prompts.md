# Visual System for IG Carousel (OpenAI Images)

## Objective
Keep visual consistency while adapting each post's unique theme.

## Baseline approach
- Use 3-5 repeatable background archetypes
- Overlay per-post theme symbols
- Keep color system stable across series

## 5 background archetypes
1. Dawn mountain mist (strategic clarity)
2. Scroll + ink desk (wisdom / planning)
3. Lantern corridor at dusk (reflection / transition)
4. River and stone path (discipline / flow)
5. Minimal temple silhouette (inner stillness)

## Master prompt shell
"Editorial illustration for personal growth content, Eastern symbolic aesthetics, calm cinematic lighting, minimal composition, textured paper feel, no text, no logo, high detail, social media friendly. Theme focus: {THEME}. Background archetype: {ARCHETYPE}."

## Theme modifiers (examples)
- Boundaries: "subtle protective circle motif, doorway threshold"
- Courage: "small figure facing vast mountain pass at sunrise"
- Focus: "single brush stroke, clear horizon line"
- Persistence: "river carving through stone over time"

## Notes on Eastern aesthetics with OpenAI
- Avoid vague phrase "Eastern ideology" alone
- Use concrete visual tokens: ink wash, bamboo, lacquer red, mountain pass, calligraphy paper texture
- Use mood words: restrained, contemplative, balanced, dignified
